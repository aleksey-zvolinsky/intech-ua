Сервер для Аульской ХПС, версии 1.0

Общее описание
Приложения для сбора информации с расходомеров на Аульской ХПС, передаваемой по средством мобильного интернета.
Приложение стартует внутри себя самостоятельный веб контейнер Jetty под управлением библиотеки Spark Web при использовании базы данных HSQLDB.
Веб контейнер по умолчанию запускается на порте 8050, со всех адресов сервера. В файле setup.ini можно изменить порт и указать один адрес сервера.

Контексты
Прием данных ведется на корневом контексте /input переменные принимаются в GET параметров запроса
По адресу /set изменяется конфигурация. Данная страница защищена авторизацией. В файле settings.ini заданы значения по умолчанию, с которыми сервер инициализируется при первом запуске.

Пользователи
При первом запуске создается два пользователя admin/admin и oper/oper. На странице конфигурации можно поменять пароль пользователю с именем oper.

Журналирование
Системное журналирование в программе осуществляется при помощи Log4J библиотеки. Настройка журналирования программы осуществляется через стандартный конфигурационный файл log4j.properties.

Дополнительная конфигурация
Конфигурационные файлы расположены в корне архива, при необходимости постоянного редактирования можно извлечь и положить в каталог том же уровне, что и jar-файл.

База данных
База данных развертывается в каталоге db. Для чистки базы нужно удалить всю папку. 
Файл db.script содержит базу в sql формате
Файл db.log содержит операции выполненые после запуска в sql формате. Содержимое данного файла переноситься в db.script файл.
